# Church Asset Tracker - Starter (MySQL + Prisma + Node + React)

This is a starter full-stack project scaffold for an asset tracker. It includes:
- React + Vite frontend
- Node + Express backend
- MySQL database via Prisma ORM
- Admin-only user creation (admin creates accounts)
- JWT authentication (login)

See server/README.md and client/README.md for local setup instructions.
